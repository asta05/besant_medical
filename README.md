# BESANT MEDICAL - CLINIC MANAGAEMENT & BILLING SYSTEM
##  OBJECTIVE

Developed a clinic management and billing system to manage medicine sales, billing operations, and customer records efficiently.

## Tech Stack

* Python (Streamlit)
* SQLite
* Pandas
* QR Code Integration (UPI)

##  Features

* Manage medicine inventory and sales
* Generate bills for customers
* Maintain customer database with order history
* Quick reorder functionality for repeat customers
* QR-based UPI payment system

##  Key Highlights

* Built a complete Point-of-Sale (POS) system for clinics
* Integrated digital payment system using QR codes
* Designed a simple and user-friendly interface using Streamlit
* Ensured efficient data handling using SQLite

## Project Structure

* `besant_medicals.py` – Main application
* `clinic_sqlite.db` – Database
* `requirements.txt` – Dependencies
* `qr_snekha.jpg` – Payment QR code

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run besant_medicals.py
